# digital_clock

## 자바스크립트 getTime()메서드를 이용한 디지털시계 프로그램입니다
### 시작을 누르면 현재시간이 뜨고 정지를 누르면 멈춥니다
### 다시 시작을 누르면 현재시각으로 시간이 진행됩니다
